<h3>내가 보려고 정리한 Git 명령어</h3>
* Download
    ```bash
    git init
    git clone <저장소 url>
  ```
  
* Upload
    ```bash
  git init
  git add .
  git commit -m "커밋 메세지"
  git remote add origin <저장소 URL>
  git push origin master
  ```      

* 기타 
  ```bash
  #현재 상태
  git status
  
  #로그
  git log
   
  #저장소 삭제
  git remote rm <저장소 URL>
  
  #git 새로운 파일을 추가
  git add <파일>
  
  #git 모든 원격 브랜치 목록 보기
  git branch -r

  #git 브랜치 생성
  git branch <브랜치 이름>
   ```
